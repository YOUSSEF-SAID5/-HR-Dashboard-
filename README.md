# -HR-Dashboard-
This project is a simple HR Dashboard that presents employee data in a clear and visual way
📊 HR Analytics Dashboard: Workforce Insights

Note: This project was created for educational purposes to practice data analysis and dashboard design. The dataset was used to demonstrate the process of organizing, analyzing, and visualizing HR data.

📊 Project Overview

This project presents an interactive HR dashboard that transforms employee data into clear and actionable insights.
The dashboard highlights workforce distribution, salaries, skills, and employee status across multiple dimensions to support better decision-making.

🛠️ Tools & Techniques

Tool: Microsoft Excel

Data Cleaning: Structured employee data for consistency and accuracy
Data Analysis: Used pivot tables to summarize employee distribution, salaries, and categories
Visualization: Built an interactive dashboard using charts, cards, and filters for better insights
💡 Key Insights & Observations
Workforce Size: The organization includes a total of 50 employees, providing a compact but diverse dataset
Gender Distribution: The workforce is nearly balanced with 52% males and 48% females, indicating good gender representation
Employment Status:
Full-time employees represent the majority, followed by contract and part-time roles, showing a stable employment structure
Salary Distribution:
Managers and Developers have the highest total salaries, reflecting their critical roles in the organization
Skills Overview:
Communication and Management are among the most common skills, highlighting the importance of soft skills alongside technical abilities
Work Location:
Employees are distributed across branch offices, head office, and remote work, with a noticeable shift toward flexible work setups
Regional Distribution:
Central and West regions show higher employee concentration compared to other regions
Age Distribution:
Most employees fall within mid-age ranges, indicating a workforce with a balance of experience and productivity
Performance Indicator:
The average rating is 2.6, suggesting potential areas for performance improvement
